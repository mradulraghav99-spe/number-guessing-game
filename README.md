# Number Guessing Game (Python)

## 📌 Project Description
This is a simple Python-based number guessing game where the computer randomly selects a number between 1 and 10. The player has 3 chances to guess the correct number.

## 🎮 Game Rules
- The computer generates a random number between **1 and 10**.
- The player gets **3 chances** to guess the number.
- If the guess is correct, the player **wins**.
- If the guess is incorrect, the game gives hints like **Too High** or **Too Low**.
- If all chances are used, the player **loses**.

## 🛠 Technologies Used
- Python
- Random module
- Loops and Conditional Statements

## ▶️ How to Run the Game
1. Download or clone this repository.
2. Open the file `guessing_game.py`.
3. Run the program using Python.
4. Enter your guesses and try to win!
